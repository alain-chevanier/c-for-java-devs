# C for Java Devs

## Lesson 1

### Topics

* Structural programming
* Hello World
* Pointers
* Arrays
* Structs
* Stack vs Heap
* Memory Allocation through `malloc`

### Usage Instructions
To compile use a command like this:

```bash
$ gcc file_name.c -o executable_name
```

To execute the produced executable file, try a command liek this:

```bash
$ ./executable_name
```
